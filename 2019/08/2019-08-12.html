proxy


简介

Proxy，代理，是ES6新增的功能，可以理解为代理器（即由它代理某些操作）

Proxy 对象用于定义或修改某些操作的自定义行为，可以在外界对目标对象进行访问前，对外界的访问进行改写。


定义

var proxy = new Proxy(target,handler)

  • target 目标对象
  • handler 一个对象，其属性是当执行一个操作时定义代理的行为的函数

注意，要实现拦截操作，必须是对proxy实例进行操作，而不是针对目标对象target进行操作


其中，proxy拦截了对象属性的get和set操作.


如果handler为{ }，则相当于proxy没有做任何拦截，访问proxy实例就相当于访问target目标对象。



proxy handler方法

  • ge(target, key, receiver) 拦截target属性读取
  • set(target, key, value, receiver) 拦截target属性的设置
  • has(target, key) 拦截target( key in proxy)的操作 并返回结果（boolean值）
  • deleteProperty(target, key) 拦截 delete proxt[key] 的操作 并返回结果（boolean值）
  • ownKeys(target) 拦截 Object.getOwnpropertyName(proxy) 、Object.keys(proxy) 、 for……in 循环。其中Object.keys()的返回结果数组中只包含目标对象自身的可遍历属性
  • getOwnPropertyDescriptor(target, key)：拦截 Object.getOwnPropertyDescriptor(proxy, key)，返回属性的描述对象
  • defineProperty(target, key, desc)：拦截Object.defineProperty(proxy, key, desc)、Object.defineProperties(proxy, descs)，返回一个 boolean 值

preventExtensions(target)：拦截Object.preventExtensions(proxy)，返回一个 boolean 值

  • getPrototypeOf(target)：拦截Object.getPrototypeOf(proxy)，返回一个对象
  • isExtensible(target)：拦截Object.isExtensible(proxy)，返回一个 boolean 值
  • setPrototypeOf(target, key)：拦截Object.setPrototypeOf(proxy, key)，返回一个 boolean 值。如果目标对象是函数，则还有两种额外操作可以被拦截
  • apply(target, object, args)：拦截 Proxy 实例作为函数调用的操作，比如proxy(...args)、proxy.call(object, ...args)、proxy.apply(...)
  • construct(target, args)：拦截 Proxy 实例作为构造函数调用的操作，比如new proxy(...args)


常用方法统计

  1. get方法用于拦截某个属性的读取操作，可以接受三个参数，依次为目标对象、属性名和 proxy 实例本身（严格地说，是操作行为所针对的对象），其中最后一个参数可选。
  2. set拦截 target 属性的设置，可以接受四个参数，依次为目标对象、属性名、value和 proxy 实例本身（严格地说，是操作行为所针对的对象），其中最后一个参数可选。
  3. 拦截 propKey in proxy 的操作，返回一个布尔值.
  4. ownKeys 拦截自身属性的读取操作。并返回目标对象所有自身属性的属性名数组

  1. apply apply 拦截 Proxy 实例作为函数调用的操作，比如函数的调用（proxy(...args)）、call（proxy.call(object, ...args)）、apply（proxy.apply(...)）等。
