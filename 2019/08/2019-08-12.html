proxy


简介

Proxy，代理，是ES6新增的功能，可以理解为代理器（即由它代理某些操作）

Proxy 对象用于定义或修改某些操作的自定义行为，可以在外界对目标对象进行访问前，对外界的访问进行改写。


定义

var proxy = new Proxy(target,handler)

  • target 目标对象
  • handler 一个对象，其属性是当执行一个操作时定义代理的行为的函数

注意，要实现拦截操作，必须是对proxy实例进行操作，而不是针对目标对象target进行操作


其中，proxy拦截了对象属性的get和set操作.


如果handler为{ }，则相当于proxy没有做任何拦截，访问proxy实例就相当于访问target目标对象。



proxy handler方法

  • ge(target, key, receiver) 拦截target属性读取
  • set(target, key, value, receiver) 拦截target属性的设置
  • has(target, key) 拦截target( key in proxy)的操作 并返回结果（boolean值）
  • deleteProperty(target, key) 拦截 delete proxt[key] 的操作 并返回结果（boolean值）
  • ownKeys(target) 拦截 Object.getOwnpropertyName(proxy) 、Object.keys(proxy) 、 for……in 循环。其中Object.keys()的返回结果数组中只包含目标对象自身的可遍历属性
